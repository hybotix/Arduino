# The Robot Smart Home Controller

### This is a small robot that has many environmental monitoring sensors as well as a complete Smart Home controller. You can also think of this little robot as a Smart Home controller on wheels! This robot will roam around its environment taking readings. If it discovers a level of contaminates or gases that is unsafe, it can send an alert to an owner. Different spots in a room can have differing readings so the robot roams around constantly.

### It can also act as a complete controller for a smart home. One thing it might do is monitor and control temperature and humidity in a room. This can be modified depending on who is in the room. There are so many possibilities!

### I am big into environmental monitoring and have been places where the air is so bad that I have had to have oxygen (thanks to a friend). I want to know the condition of my environment at all times. I want to make sure my environment is always set to be optimum for my health whether at night or during the day.
